## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/felipealvescosta/mobile-toast
```

2. Install packages

```sh
yarn install
```

3. Run Project

```sh
yarn ios or yarn android
```
